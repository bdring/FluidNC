** FluidNC installation instructions for Windows

1. Download the file InstallFluidNC.ps1 from the release assets.

2. Start a PowerShell window on your Windows computer

3. Run the following PowerShell command to enable the InstallFluidNC.ps1 to run

    Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

4. Determine the version and build type that you wish to run.
   The version is something like  v3.0.5
   The build type is either  wifi , bt , wifibt , or noradio

5. Run the following PowerShell command substituting your chosen version and type:

    ./InstallFluidNC.ps1 -version v3.0.5 -type wifi

  It will fetch the release archive for that version and type, unzip it, fetch
  and unzip the "esptool" program, and run that program to install FluidNC on
  your ESP32 - which must be plugged in to a USB port.
